# Open Shifts / VOT Foundation

Signal Schedule v2.3.0 introduces the Open Shifts / Voluntary Overtime Foundation.

## Purpose

This release reserves and previews the workflow for posting open shifts, collecting voluntary overtime interest, showing staffing impact, and preparing future supervisor/admin award decisions.

## Included

- Open shift preview records
- VOT request preview records
- Admin-controlled request reason seed options
- Preferred shift and assignment concepts
- Interest level concepts
- Contact preference concepts
- Qualification matching placeholders
- Staffing impact placeholders
- Full-shift and partial-shift hour calculation preview
- Admin override planning

## Important Rule

Default request reasons are seed options only. Admins must be able to add, edit, disable, delete, and control employee visibility for request reasons before production use.

## Not Active Yet

- No live submission
- No award workflow
- No employee login
- No notifications
- No payroll export
- No equalization engine
- No mandation logic
