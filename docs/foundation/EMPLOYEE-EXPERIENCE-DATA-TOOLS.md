# Signal Schedule v5.1.0 — Employee Experience & Data Tools

v5.1.0 consolidates the next roadmap step into one milestone: employee experience, profile management, templates, imports, and exports.

## Principles

- Admin = Employee + More.
- Employee, supervisor, admin, and system admin roles inherit the same self-service pages.
- One agency has one living schedule with many snapshots.
- Data tools prepare the system for real onboarding without forcing manual entry forever.

## Included foundation

- Employee dashboard expansion for My Schedule, My Requests, My Availability, My Profile, Notifications, and data tools.
- New Data Tools page for templates, imports, exports, validation, duplicate detection, and migration readiness.
- Migration 045 for persistent schedule templates, import jobs, export jobs, employee profile extras, and employee notification preferences.
- Data contracts and preview data for the employee experience/data tools foundation.

## Not included yet

- Full authentication.
- Real file upload processing.
- Excel generation.
- Request approvals.
- Time clock.

Those remain future milestones.
