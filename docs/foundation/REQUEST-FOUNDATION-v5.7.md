# Request Foundation v5.14.2

Signal Schedule v5.14.2 adds request foundation structures before the full v6 Request Engine.

## Scope

- Time off request foundation
- Voluntary overtime request foundation
- Request status board foundation
- Partial-day hour range handling
- Full-day scheduled-shift handling
- Agency request increment rules
- Admin override visibility
- Admin-controlled reason presets
- Request history/audit foundation

## Foundation table targets

- `schedule_requests`
- `schedule_request_types`
- `schedule_request_status`
- `schedule_request_reasons`
- `schedule_request_history`

No live database migration is required for v5.14.2.
