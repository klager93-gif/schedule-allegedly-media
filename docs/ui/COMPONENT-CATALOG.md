# UI Component Catalog — v5.15.0

Signal Schedule v5.15.0 introduces an internal component catalog at `/pages/admin/ui-catalog.html`.

## Purpose

The catalog standardizes Schedule-only interface patterns before larger operational modules are expanded.

## Covered patterns

- Page headers
- Action bars
- Filter bars
- Buttons
- Cards
- Badges
- Form controls
- Tables
- Alerts
- Empty states
- Modal previews

## Rule

Future Signal Schedule screens should reuse these primitives before adding page-specific component styles. The catalog must stay isolated from other Allegedly Media or Signal Labs projects.
