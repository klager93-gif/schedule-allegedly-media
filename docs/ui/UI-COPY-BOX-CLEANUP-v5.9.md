# Signal Schedule v5.10.0 — UI Copy Box Cleanup

A previous request for copy-friendly ChatGPT handoff formatting was mistakenly implemented as in-app copy buttons and code-style boxes across foundation pages.

## v5.9 rule

Signal Schedule UI should look like software, not a release note or developer scratchpad.

Use normal cards, tables, status chips, and labels for:

- request types
- route names
- feature lists
- page descriptions
- status summaries
- preview IDs

Reserve code blocks/copy affordances for truly technical screens, such as SQL examples, API contracts, or export payloads.
