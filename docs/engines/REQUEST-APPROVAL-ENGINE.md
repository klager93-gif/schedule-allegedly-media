# Signal Schedule v5.1.0 — Request & Approval Engine

Signal Schedule v5.1.0 turns the earlier request preview pages into a unified Request & Approval Engine foundation.

## Core principle

Requests are operational events that can affect the living agency schedule, coverage, staffing, leave banks, overtime, trades, and employee history.

## Included request types

- Vacation
- Sick
- Comp time
- Training
- Shift trades
- Voluntary overtime
- Open shift interest
- Profile/data change requests

## Workflow states

- Draft
- Submitted
- Under review
- Approved
- Denied
- Cancelled
- Applied to schedule

## Foundation added

- Central Requests page
- Request queue preview
- Approval workflow preview
- Policy check preview
- Decision/audit trail preview
- Migration 046 for request cases, decisions, events, and comments
- API preview contract for request engine data

## Not yet included

- Employee authentication
- Production request submission
- Production approval writes
- Payroll export integration
- Notification delivery

Those remain future work after the foundation is validated.
