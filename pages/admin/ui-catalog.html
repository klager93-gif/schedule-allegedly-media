Usage Rules

  • Use Primary buttons only for the main page action.
  • Use badges for state, not decoration.
  • Use action bars above tables and filter bars directly below page headers when filters are needed.
  • Keep admin/internal screens inside the Signal Schedule shell and avoid shared styling from other projects.
