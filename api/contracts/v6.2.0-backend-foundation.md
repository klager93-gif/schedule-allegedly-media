# API Contract Draft — v6.2.0 Backend Foundation

## Health

`GET /api/health`

Returns application and database status.

## Auth

`POST /api/auth/login`

Request:

```json
{ "email": "admin@example.com", "password": "ChangeMe123!" }
```

Response:

```json
{ "user": { "id": "...", "email": "...", "name": "...", "agency": "...", "roles": [] } }
```

`POST /api/auth/logout`

Returns `{ "ok": true }`.
