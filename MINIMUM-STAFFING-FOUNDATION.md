# Schedule v2.0.0 — Minimum Staffing Foundation

This release adds the foundation for defining minimum staffing requirements by assignment, role, shift, location, and day.

## Added

- Minimum staffing template seed data
- Minimum staffing preview seed data
- JSON adapter boundary
- Repository boundary
- Service boundary
- Read contract
- Read-only Coolify API route
- Future Postgres schema
- Frontend preview section

## Not Added

- No staffing editor
- No leave approval logic
- No VOT bidding logic
- No mandation engine
- No schedule generator
- No production writes for minimum staffing
