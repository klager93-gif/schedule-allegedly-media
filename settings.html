Signal Schedule v5.2.0

Admin Settings Sandbox

Configure agency, time, shift, coverage, and request-rule values. In this release these settings are preview/local data; the builder uses the same settings file for validation.

Open Schedule Builder

Settings now represented

  • Agency name, timezone, week start, pay period start, and time format.
  • Shift labels, start/end times, paid hours, roles, and shift groups.
  • Minimum/maximum coverage requirements by role and shift.
  • Time-off and overtime increment rules with future admin override support.

Next database step

v4.1 proves the admin settings model in the browser. A later release should persist these values to Postgres and scope them by agency.
